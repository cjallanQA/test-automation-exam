# Manual Test Cases

## TC-01 Drag and Drop

Objective:
Check if box A and box B switch positions.

Steps:
1. Open homepage.
2. Open Drag and Drop.
3. Drag box A to box B.

Expected:
Box headers are swapped (left B, right A).

## TC-02 Infinite Scroll

Objective:
Check initial content and loading of extra content while scrolling.

Steps:
1. Open homepage.
2. Open Infinite Scroll.
3. Confirm text is visible on load.
4. Scroll down to load more content.

Expected:
Additional block appears and last loaded block is visible.

## TC-03 Dynamic Content

Objective:
Check which rows change after refresh.

Steps:
1. Open homepage.
2. Open Dynamic Content.
3. Record row text.
4. Refresh page.
5. Compare rows.

Expected:
At least one row changes after refresh.

## TC-04 Horizontal Slider

Objective:
Set slider to 3.5 and verify value.

Steps:
1. Open homepage.
2. Open Horizontal Slider.
3. Move slider using arrow keys.
4. Stop at 3.5.

Expected:
Displayed slider value is 3.5.
